数据结构-数组

2019-11-03 java-data 算法 评论 字数统计: 1.6k(字) 阅读时长: 7(分)

引言

线性表 - 数组和矩阵

概念

数组是一种连续存储线性结构，元素类型相同，大小相等，数组是多维的，通过使用整型索引值来访问他们的元素，数组尺寸不能改变。

优点

  • 查找、存取速度快

缺点

  • 事先必须知道数组的长度
  • 插入删除元素很慢
  • 空间通常有限制的
  • 需要大块连续的内存块
  • 插入删除元素的效率很低

数组和矩阵相关题目

118.杨辉三角(easy)

输入：5
[
     [1],
    [1,1],
   [1,2,1],
  [1,3,3,1],
 [1,4,6,4,1]
]

分析：

一步一步分析数组的规律，每一行的第一个和最后一个都是1，前两行不需要处理中间过程，而中间过程无非是当前索引的值和上一个索引的值相加得到新的value

class Solution {
    public List<List<Integer>> generate(int numRows) {
        List<List<Integer>> ans = new ArrayList<>();
        for(int i = 0; i < numRows; i++) {
            List<Integer> curRow = new ArrayList<>();
            for(int j = 0; j <= i; j++) {
                  // 处理第一个元素和最后一个元素
                if(j == 0 || j == i) {
                    curRow.add(1);
                    continue;
                }
                  // 第一行和第二行不需要处理中间过程
                if(i == 0 || i == 1) {
                    continue;
                }
                  // 处理中间过程
                List<Integer> preRow = ans.get(i - 1);
                int value = preRow.get(j - 1) + preRow.get(j);
                curRow.add(value);
            }
            ans.add(curRow);
        }
        return ans;
    }
}

26.删除排序数组中的重复项(easy)

不要使用额外的数组空间，你必须在原地修改输入数组并在使用 O(1) 额外空间的条件下完成。
给定数组 nums = [1,1,2], 
函数应该返回新的长度 2, 并且原数组 nums 的前两个元素被修改为 1, 2。 
你不需要考虑数组中超出新长度后面的元素。

分析：前提是已经排序，给定一个指针p，每次和该指针p进行比较，如果不相等，则指针+1，将不相等的值放进去，如果相等，则重复，不用管。

class Solution {
    public int removeDuplicates(int[] nums) {
        int p = 0;
        for(int i = 1; i < nums.length; i++) {
            if(nums[p] != nums[i]) {
                nums[++p] = nums[i];
            }
        }
        return p+1;
    }
}

66.加一(easy)

输入: [1,2,3]
输出: [1,2,4]
解释: 输入数组表示数字 123。

分析：因为加1，carry初始值为1，每次和进位carry相加，临界值10，判断情况，如大于10，carry=1，否则carry=0。最后考虑全进位，数组多一位的问题。

class Solution {
    public int[] plusOne(int[] digits) {
        int length = digits.length;
        int[] res = new int[length + 1];
        int carry = 1;
        for (int i = length - 1; i >= 0 ; i--) {
            int sums = digits[i] + carry;
            if (sums >= 10) {
                carry = 1;
                res[i] = sums % 10;
            } else {
                carry = 0;
                res[i] = sums;
            }
        }
        if (carry == 1) {
            res[0] = 1;
            return res;
        }
        return Arrays.copyOfRange(res,0,length);

    }
}

88.合并两个有序数组(easy)

输入:
nums1 = [1,2,3,0,0,0], m = 3
nums2 = [2,5,6],       n = 3

输出: [1,2,2,3,5,6]

分析：两种情况，m=0和m!=0的情况，m=0就是nums2，m!=0，就要比较了，多设个额外指针p。目的是存放比较后的最大值，因为nums1和nums2是排序的，所以比较的思路是nums1和nums2的从右到左的最大值。注意越界问题

class Solution {
    public void merge(int[] nums1, int m, int[] nums2, int n) {
          //第一种情况
        if(m == 0) {
            for(int i = 0; i < n; i++){
                nums1[i] = nums2[i];
            }
            return;
        }
          // 第二种情况
        int p = nums1.length - 1;
        int a1 = m - 1;
        for(int i = n - 1; i >= 0; i--) {
            while(a1 >= 0 && nums1[a1] > nums2[i]) {
                nums1[p--] = nums1[a1--];
            }
            nums1[p--] = nums2[i];
        }
    }
}

121.买卖股票的最佳时间(easy)

输入: [7,1,5,3,6,4]
输出: 5
解释: 在第 2 天（股票价格 = 1）的时候买入，在第 5 天（股票价格 = 6）的时候卖出，最大利润 = 6-1 = 5 。
     注意利润不能是 7-1 = 6, 因为卖出价格需要大于买入价格。
输入: [7,6,4,3,1]
输出: 0
解释: 在这种情况下, 没有交易完成, 所以最大利润为 0。

分析：两次循环比较，这是最简单的方法，最快的还是动态规划。。等动态规划在写这道题

class Solution {
    public int maxProfit(int[] prices) {
        int proMax = 0;
        for(int i =  0; i < prices.length; i++){
            for(int j = prices.length - 1; j > i; j--){
                if(prices[i] < prices[j]) {
                    proMax = Math.max(proMax, prices[j] - prices[i]);
                }
            }
        }
        return proMax;
    }
}

122 买卖股票的最佳时间2(easy)

分析：可用贪心思想

class Solution {
    public int maxProfit(int[] prices) {
        int ans = 0;
        for(int i = 1; i < prices.length; i++){
            if(prices[i] > prices[i - 1]) {
                ans += prices[i] - prices[i - 1]; 
            }
        }
        return ans;
    }
}

169 求众数(easy)

输入: [3,2,3]
输出: 3
输入: [2,2,1,1,1,2,2]
输出: 2

分析：相同count++；遇到不同，则–，最后总有一个是多出来的

class Solution {
    public int majorityElement(int[] nums) {
        int count = 1;
        int ans = nums[0];
        for(int i = 1; i < nums.length; i++) {
            if(nums[i] == ans) {
                count++;
            } else {
                count--;
                if(count == 0) {
                    ans = nums[i + 1];
                }
            }
        }
        return ans;
    }
}

189. 旋转数组(easy)

输入: [1,2,3,4,5,6,7] 和 k = 3
输出: [5,6,7,1,2,3,4]
解释:
向右旋转 1 步: [7,1,2,3,4,5,6]
向右旋转 2 步: [6,7,1,2,3,4,5]
向右旋转 3 步: [5,6,7,1,2,3,4]

分析：两种方法

/**
* 双重循环
* 时间复杂度：O(kn)
* 空间复杂度：O(1)
*/
class Solution {
    public void rotate(int[] nums, int k) {
        int n = nums.length;
        k %= n;
        for(int i = 0; i < k; i++){
            int temp = nums[n - 1];
            for(int j = n - 1; j >0; j--) {
                nums[j] = nums[j - 1];
            }
            nums[0] = temp;
        }
    }
}
/**
* 翻转
* 时间复杂度：O(n)
* 空间复杂度：O(1)
*/
class Solution {
    public void rotate(int[] nums, int k) {
        int n = nums.length;
        k %= n;
        reverse(nums, 0, n - 1);
        reverse(nums, 0, k - 1);
        reverse(nums, k, n - 1);
    }
    private void reverse(int[] nums, int start, int end) {
        while(start < end) {
            int temp = nums[start];
            nums[start++] = nums[end];
            nums[end--] = temp;
        }
    }
}

217.存在重复元素(easy)

输入: [1,2,3,1]
输出: true

输入: [1,2,3,4]
输出: false

分析：想了想，还是用哈希吧，循环太慢了

class Solution {
    public boolean containsDuplicate(int[] nums) {
        HashMap<Integer, Integer> map = new HashMap<>();
        for (int num : nums) {
            if (map.containsKey(num)) {
                return true;
            } else {
                map.put(num, 1);
            }
        }

268.缺失数字(easy)

输入: [3,0,1]
输出: 2
输入: [9,6,4,2,3,5,7,0,1]
输出: 8

分析：异或即可

class Solution {
    public int missingNumber(int[] nums) {
        int ans = nums.length;
        for(int i = 0; i < nums.length; i++) {
            ans ^= nums[i];
            ans ^= i;
        }
        return ans;
    }
}

283.移动零(easy)

输入: [0,1,0,3,12]
输出: [1,3,12,0,0]

分析：用index指针记录，再从index之后补0

class Solution {
    public void moveZeroes(int[] nums) {
        int index = 0;
        for(int i = 0; i < nums.length; i++) {
            if(nums[i] != 0) {
                nums[index++] = nums[i];
            }
        }
        for(int i = index; i < nums.length; i++) {
            nums[i] = 0;
        }
    }
}
